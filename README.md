![sample](/docs/sample.png)

アイデア元:
https://twitter.com/e869120/status/1519310341024677888

## 灰色の細分化について

内部レート 0 以上を灰色、0 未満を白抜きとして表示しています。

灰色は 100 ごと、白抜きは 200 ごとにアイコンが変化します。
